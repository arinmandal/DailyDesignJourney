CSS Display Properties

The CSS `display` property is used to specify the layout of an element. It controls whether the element will be displayed as a block, inline, inline-block, or not at all.

Block Elements

display: block;

Block elements take up the full width available and start on a new line. They can have margins and padding applied to both sides, as well as top and bottom.

Inline-Block Elements

display: inline-block;

Inline-block elements combine properties of both block and inline. They take up as much width as necessary but do not start on a new line. They can have margins and padding applied to both sides, top, and bottom.

None Display

display: none;

The `none` display value hides the element from being displayed at all. The space it would occupy is not reserved.

This is often used for conditional display or hiding elements that are not relevant under certain conditions.
