What is the CSS Box Model?

The CSS box model is a fundamental concept that describes how all HTML elements are structured. Every element on a web page can be thought of as a rectangular box with four distinct areas:

  • Content - The actual content of the element (text, images, etc.)
  • Padding - The space between the content and the border
  • Border - The line that surrounds the padding and content
  • Margin - The space outside the border that separates elements from each other

How Box Sizing Works

By default, CSS uses the content-box model where width and height only apply to the content area. When you add padding and border, the total size increases.

In this example, we've set box-sizing: border-box; globally, which means that the width and height include padding and border, making layout calculations much more predictable.

Visualizing Box Model Components

Each element has these measurable properties:

Property Description Example Value
Width Horizontal dimension of the content area 200px
Height Vertical dimension of the content area 200px
Padding Space inside the border 20px
Border Line around the padding and content 1px solid red
Margin Space outside the border 50px

Total Width Calculation (with border-box):

200px (width) + 20px (padding left) + 20px (padding right) + 1px (border left) + 1px (border right) = 242px total width

Total Height Calculation (with border-box):

200px (height) + 20px (padding top) + 20px (padding bottom) + 1px (border top) + 1px (border bottom) = 242px total height
